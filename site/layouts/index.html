Features

Everything you need to route messages across any network.

Policy-Driven Routing

AccessEvaluator rules engine with source/dest filtering, rate limiting, and failover groups.

Transform Pipeline

Per-message transforms: zstd compression, SMAZ2 lossless, MSVQ-SC lossy semantic, AES-256-GCM encryption.

USB Hot-Swap

DeviceSupervisor with VID:PID identification, automatic protocol probing, claim-based port management.

Pass Scheduling

SGP4 TLE-driven satellite pass prediction with 4-mode state machine for power-efficient operation.

Dead Letter Queue

ISU-aware backoff, automatic retry with exponential backoff, no message loss.

Multi-Tenant Hub

Hub SaaS platform with OAuth2/OIDC, RBAC, API keys, tenant isolation, NATS/MQTT bus.
